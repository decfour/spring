🛒 Commerce Service
> **Spring MVC를 활용한 이커머스 핵심 기능 구현 프로젝트**

**메모리 저장소에서 SQL 데이터베이스 연동**으로 고도화하는 과정을 담고 있습니다.

---

### 🚀 Key Objectives
* **MVC Pattern:** 계층형 아키텍처(Controller-Service-Repository)의 분리와 이해
* **Data Persistence:** 비즈니스 로직의 데이터를 **H2 Database**에 영속화 (진행 중)
* **Practical Application:** 서블릿/JSP 원리를 스프링 부트 환경에 실전 적용

### 🛠 Tech Stack
* **Framework:** Spring Boot 3.x
* **Language:** Java 17
* **Database:** H2 (Embedded) → MySQL (Planned)
* **Build:** Gradle

### 📌 Main Features
1. **계층별 책임 분리:** 스프링 빈 등록을 통한 의존관계 주입(DI) 활용
2. **데이터 영속화:** JDBC/JPA를 이용한 SQL 연동 및 데이터 관리
3. **상품 관리 로직:** MVC 기반의 CRUD 기능 구현
