🛒 Commerce Service
> **Spring MVC를 활용한 이커머스 핵심 기능 구현 프로젝트**

---

### 🚀 Key Objectives
* **MVC Pattern:** 계층형 아키텍처(Controller-Service-Repository) 분리와 이해
* **Data Persistence:** 비즈니스 로직의 데이터를 MYSQL에 영속화 (Planned)
* **Practical Application:** Thymeleaf 템플릿 엔진 적용

### 🛠 Tech Stack
* **Framework:** Spring Boot 3.x
* **Language:** Java 17
* **Database:** MySQL (Planned)
* **Build:** Gradle

### 📌 Main Features
1. **계층별 책임 분리:** 스프링 빈 등록을 통한 DI 활용
2. **데이터 영속화:** JPA로 MYSQL 연동 및 데이터 관리
3. **상품 관리 로직:** MVC 기반의 CRUD 기능 구현
